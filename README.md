# Overview
## CLA 2026 Crash Course: Statistical Methods for Linguistic Data 

The right statistical model depends on your data. In this hands-on workshop, linguists will learn about methods to analyze some common types of data in the field, e.g., acceptability judgments, corpus counts, acoustic measurements, etc. We will cover the full pipeline from identifying variable types to exploring data through visualization, model selection, fitting, and interpretation of results. Along the way, we will tackle the common pitfalls that trip up even experienced researchers. Workshop attendees will walk away with practical tools and reference materials to apply to their own analyses. This workshop includes executable code but is suitable for all experience levels.
